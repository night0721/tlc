# tlc 
**t**iny **l**ol**c**at is a fun program to display text in rainbow colors

# Usage
```
fortune | cowsay | tlc
cat <x> <y> <z> | tlc
```

# Building
You will need to run these with elevated privilages.
```
$ make
# make install
```
